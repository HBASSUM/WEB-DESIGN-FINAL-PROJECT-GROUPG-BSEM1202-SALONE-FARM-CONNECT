#  Salone Farm Connect

##  Project Overview
Salone Farm Connect is a web-based digital platform developed to address challenges faced by farmers in Sierra Leone. Many farmers struggle to access current market prices, reliable farming information, and direct connections to buyers. This often results in low profits and limited market opportunities.

The platform provides real-time market price information, farming guidance, buyer connection services, and farmer registration features. By using responsive web technologies, the system is accessible on desktops, tablets, and mobile phones, allowing farmers across Sierra Leone to benefit from digital agricultural services.

This project supports **Digital Public Good principles** by promoting access to information, economic growth, and sustainable agricultural development.
